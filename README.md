# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1aCynNb3s58ENqZ7oYXQrF5Xi7tDZKEzA

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Notes
- This app runs fully offline with local lesson packs.
- For speaking and listening features, your browser should support Speech Synthesis and Speech Recognition (Chrome-based browsers work best).
